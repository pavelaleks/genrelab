# Папка для изображений

Если вы хотите использовать локальное изображение Самарканда вместо изображения из интернета:

1. Поместите файл изображения в эту папку с именем `samarkand.jpg`
2. Поддерживаемые форматы: JPG, PNG, GIF
3. Рекомендуемый размер: 400x300 пикселей или больше (будет автоматически масштабировано)

Если файл `samarkand.jpg` не найден, приложение автоматически использует изображение Регистана из Wikipedia.
