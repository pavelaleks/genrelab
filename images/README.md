# Папка для изображений

Эта папка предназначена для хранения изображений, используемых в приложении NARRALAB.

## Поддерживаемые форматы
- JPG/JPEG
- PNG
- GIF
- WebP

## Рекомендации
- Рекомендуемый размер: минимум 400x300 пикселей
- Изображения будут автоматически масштабироваться при необходимости
- Оптимизируйте изображения для веб-использования для лучшей производительности
